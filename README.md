# Bank Management System (C)

## Description
This project is a simple bank management system developed using C language.
It allows users to create accounts, deposit and withdraw money, and store data using file handling.

## Features
- Create account
- Display all accounts
- Deposit money
- Withdraw money
- File handling using .dat file

## How to Run
1. Compile the program using any C compiler
2. Run the program
3. Choose options from the menu
